修改centos7网卡名称

首先进入网卡配置文件，先把文件里的名字改过来
vi /etc/sysconfig/network-scripts/ifcfg-eno32
保存退出
cd /etc/sysconfig/network-scripts/
mv ifcfg-eno32 ifcfg-eth0

修改grub配置文件/etc/grub2.cfg，把net.ifnames=0 biosdevname=0参数 加入指定位置
要加的位置 menuentry 下 linux 16 rhgb quiet 之前

vim /etc/default/grub

运行命令grub2-mkconfig -o /boot/grub2/grub.cfg 来重新生成GRUB配置并更新内核参数。

reboot 重启

更改uuid&查看服务器信息

更改uuid

在客户端新增一个设备，得到新的uuid
使用之前的uuid登录服务器，修改/etc/salt/minion_id文件，删除里面原有的uuid并加入新的uuid，重启cgroupv2（systemctl restart cgroupv2）
然后清理hyp的缓存 hyp --clear，接收这个uuid hyp -l $uuid -dst a
然后就可以使用uuid登录进去了

容器部署常用的命令

某个之前部署好的机器重新部署为host模式

先用del函数删除拨号，然后分盘更新容器

分盘更新容器

curl -sSL https://automateplus.oss-accelerate.aliyuncs.com/container-partition.sh | bash -s -- -u {pcdn/vdp/cp2p/p2p/ksp2p} $version

其他的分盘命令

分盘部署

curl -sSL https://automateplus.oss-accelerate.aliyuncs.com/container-partition.sh | bash -s -- -c {pcdn/vdp/cp2p/p2p/ksp2p} $num $version

分盘删除

curl -sSL https://automateplus.oss-accelerate.aliyuncs.com/container-partition.sh | bash -s -- -d {pcdn/vdp/cp2p/p2p/ksp2p} $num

分盘删除某个盘上的所有容器

curl -sSL https://automateplus.oss-accelerate.aliyuncs.com/container-partition.sh | bash -s -- -q $盘符名称

pip安装本地包

如果pip下载速度极低则可以从官网上下载本地安装包安装python库

pip install *.whl 直接在pip install命令后添加whl包的全路径名就能本地安装成功了
下载需要的包，一般为zip、tar.gz等的压缩包，解压后，打开命令行，进入解压目录，使用python setup.py install命令安装。
